Understanding the CSS Box Model

Every element in web design is a rectangular box

  • Content
  • Padding
  • Border
  • Margin

Content Area

This is the actual content of the box, where text and images appear.

The size is determined by the width and height properties.

Padding

Padding creates space around the content, inside the border.

It helps separate content from the border.

Border

The border surrounds the padding (if any) and content.

It can be styled with different colors, widths, and styles.

Margin

Margin creates transparent space outside the border.

It separates the element from other elements on the page.
